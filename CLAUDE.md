# 动态几何视频项目 - 学思记

## 项目概述
生成初中几何模型的动态演示视频，用于AI教学。

## 技术栈
- **动画引擎**: Manim (3Blue1Brown) - `/usr/bin/python3` (Python 3.9)
- **TTS配音**: gTTS / edge-tts
- **字幕**: ffmpeg 烧录
- **输出格式**: MP4 (H.264)

## 制作流程
1. 确定演示脚本（关键帧 + 讲解文案）
2. Manim代码生成动画
3. TTS配音合成
4. ffmpeg字幕烧录 + 合并
5. 质量审核
